# 2048 Game

This is a simple implementation of the popular game 2048, built with React.

## Features

- Smooth and responsive gameplay
- Score tracking
- Colorful UI

## Installation

First, clone the repository:

```bash
git clone https://github.com/lutif/2048.git
```

Then, install the dependencies:

```bash
npm install
```
## Running the app

```bash
npm start
```
